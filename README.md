# IggyShop
## The shop app developed with following features:
- database
  - sign up
  - login
- email validation
- API
- navigation
- changing avatars

## made with:
- Android
- Kotlin
- Coroutines
- Flow
- Dagger
- MVVM
- Jetpack Compose
- Clean architecture
- Retrofit (api)
- Room (database)
- Multimodularity
